# PROPOSTA COMERCIAL: CMS - Painel Administrativo

## Murayama Engenharia

---

## 📋 RESUMO EXECUTIVO

Desenvolvimento de um **Painel Administrativo (CMS)** para gerenciamento centralizado de conteúdo do site da Murayama Engenharia. Sistema seguro com autenticação e dois níveis de acesso (Master e Admin), permitindo editar obras, serviços, depoimentos, parceiros e demais dados do site sem necessidade de código.

---

## 🎯 O QUE SERÁ POSSÍVEL FAZER

### Através do Painel Admin, você poderá:

✅ **Gerenciar Portfólio de Obras**

- Adicionar/editar/remover projetos
- Upload de múltiplas imagens
- Organizar por categoria
- Adicionar descrições detalhadas
- Publicar/Despublicar

✅ **Atualizar Serviços**

- Editar descrições
- Reorganizar apresentação
- Ativar/desativar serviços

✅ **Gerenciar Depoimentos**

- Adicionar depoimentos de clientes
- Ordenar por importância
- Ativar/desativar exibição

✅ **Manter Parceiros Atualizados**

- Adicionar novos parceiros
- Atualizar logos
- Remover antigos

✅ **Administrar Dados de Contato**

- Telefones, emails
- Endereço
- Redes sociais
- Mensagens WhatsApp

✅ **Monitorar Atividades**

- Ver quem fez alterações
- Quando foram feitas
- Reverter mudanças (opcional)

---

## 👥 PERFIS DE ACESSO

### **MASTER** (Proprietário/CEO)

- Acesso total ao sistema
- Criar/gerenciar usuários
- Alterar permissões
- Visualizar logs completos
- Configurações avançadas

### **ADMIN** (Gestor de Conteúdo)

- Gerenciar obras, serviços, depoimentos
- Adicionar parceiros
- Atualizar contatos
- Editar conteúdo de páginas
- ❌ Não pode: deletar dados, gerenciar usuários

1. **Dashboard** - Overview de atividades e estatísticas
2. **Gerenciador de Obras** - CRUD completo com galeria
3. **Gerenciador de Serviços** - Edição e reordenação
4. **Gerenciador de Depoimentos** - CRUD com ativação/desativação
5. **Gerenciador de Parceiros** - Upload de logos

7. **Gerencimento de Usuários** - Apenas para Master
8. **Logs de Auditoria** - Rastreamento de mudanças

## 🔒 SEGURANÇA & CONFORMIDADE

✅ Autenticação por JWT  
✅ Criptografia de senhas (bcrypt)  
✅ HTTPS/SSL  
✅ Backup automático do banco de dados  
✅ Rastreamento de todas as mudanças (auditoria)  
✅ Proteção contra SQL injection e XSS

---

## 📱 Tecnologias Utilizadas

- **Frontend:** React 18 + TypeScript (compatível com site atual)
- **Backend:** Node.js + Express
- **Banco:** PostgreSQL (nuvem)
- **Storage:** AWS S3 ou Vercel Blob
- **Hospedagem:** Vercel ou AWS
- **Segurança:** JWT + bcrypt + OWASP
